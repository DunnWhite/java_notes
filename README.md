# Java学习笔记合集

## 项目概述

这是一个在2023年学习Java编程过程中整理的完整笔记合集。笔记内容系统全面，涵盖了从Java基础语法到高级特性的知识体系，适合Java初学者和有一定基础的学习者参考使用。

## 内容范围

笔记内容按照学习进度组织，主要包括以下主题：

- **数组与二维数组**：数组的基本操作、多维数组应用
- **面向对象编程**：类与对象、封装、继承、多态等核心概念
- **异常处理**：Java异常体系、自定义异常、异常处理最佳实践
- **常用类库**：String、包装类、日期时间类等核心API的使用
- **集合框架**：List、Set、Map等集合类的原理与使用
- **泛型编程**：类型安全、泛型类、泛型方法
- **IO流操作**：文件读写、序列化、NIO基础
- **综合项目实践**：通过完整项目巩固所学知识

## 学习来源

本笔记的学习内容主要参考**韩顺平老师**的Java公开课程，进度安排和案例设计与原课程保持一致。笔记在理解课程内容的基础上进行了系统化整理和补充，形成了结构化的知识体系。
